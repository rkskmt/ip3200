NumPyとは

  • NumPyは数値計算のデファクトスタンダード
    • 普通、数値計算やデータ処理にlistは使わない
  • 内部的にはC言語で実装されている
    • 通常のリストに比べて非常に高速

インストールと使用

ターミナルからインストール

pip install numpy

冒頭でimportして使用（ただしnpというエイリアス指定が一般的）

import numpy as np

inportの as とは？

import numpy as np の as np は別名（エイリアス）をつける構文

import numpy         # 使用時にnumpy.array(...) と書く
import numpy as np   # np.array(...) でよいので短い
  • 注：np の部分は何でもよく別の名前も使用可能
    • ただし、Numpyは np が非常に一般的

NumPy配列（ndarray）

  • 数値のリストっぽいものをnpに渡すとndarrayになる
  • ndarray は list によく似たNumPy用の配列型
    • 先頭の nd は多次元配列（N-Dimensional Array）の略
  • NumPyはndarrayを効率よく処理するためのライブラリ
import numpy as np

arr = np.array([1, 2, 3, 4]) # 1-4のndarray

# 以下でarrを数値計算（例：平均・分散 etc.）

ノート

np.ndarray() ではなく np.array() と書くことに注意

基本的な初期化

リストを np.array() に渡すと、NumPy配列になります。

import numpy as np

list = [1, 2, 3, 4]
arr = np.array(list)

arr = np.array([1, 2, 3, 4])    # 同じ
arr = np.array(range(1, 5))     # 同じ。実際には後述のarangeがよく使われる

list += list
arr = np.array(list)

初期化（ヘルパー関数）

  • 下記のよく使う数列は、ヘルパー関数で一撃生成
  • 特にrandom 系は強力
    • テスト用データを作るときに非常によく使われる
import numpy as np

np.zeros(3)          # 0.0が3つの要素で初期化。 [0. 0. 0. ] 
np.ones(5)           # 1.0が5つの要素で初期化。 [1. 1. 1. 1. 1.]
np.full(4, 7)        # 7が4つの要素で初期化。 [7 7 7 7]

# arrangeではなくarange（"a range" = ある範囲）
np.arange(4)         # [0 1 2 3]
np.arange(1, 5)      # [1 2 3 4]
np.arange(1, 10, 2)  # [1 3 5 7 9]

np.random.rand(3)           # [0.0, 1.0) の一様乱数 3要素
np.random.randint(10)       # 0〜9のランダムな整数
np.random.randint(1, 7, 3)  # 1〜6のランダムな整数 3要素
np.random.shuffle(arr)      # 配列をランダムに並べ替え（破壊的）

要素へのアクセス

  • アクセス方法はlistと同じ
  • スライスにも対応
import numpy as np

arr = np.array([1, 2, 3, 4])

print(arr[0])    # 先頭の要素（1）
print(arr[1])    # 2番目の要素（2）
print(arr[-1])   # 最後の要素（4）
arr[0] = 10      # 1番目の要素を10に変更

# スライス
print(arr[1:3])  # 2番目から3番目まで（[2, 3]）
print(arr[:2])   # 先頭から2番目まで（[1, 2]）
print(arr[2:])   # 3番目から最後まで（[3, 4]）

型

  • ndarray全体は numpy.ndarray
  • 各要素は代入時のリテラルによって決まる
    • intなら numpy.int64
    • floatなら numpy.float64
  • 異なる型の混在はできない
import numpy as np

arr = np.array([1, 2, 3, 4])
print(type(arr))    # <class 'numpy.ndarray'>
print(type(arr[0])) # <class 'numpy.int64'>

arr2 = np.array([1.0, 2.0, 3.0, 4.0])
print(type(arr2))    # <class 'numpy.ndarray'>
print(type(arr2[0])) # <class 'numpy.float64'>

強力な演算

  • Listと違い、各要素に対する演算が可能
    • for文が不要なため非常に高速
import numpy as np

my_array = np.array([1, 2, 3])
my_array * 2  # array([2, 4, 6])  ← 要素ごとの計算

# listに * 2 を使うと...
print(my_list * 2)  # [1, 2, 3, 1, 2, 3]  ← 演算ではなく、２つのリストのconcat

# listで同じことをするにはfor文や内包表記が必要
my_list = [1, 2, 3]
[x * 2 for x in my_list]

その他（長さやアクセス方法）

  • listと同じくlenもある
    • ただし、sizeを使うことが多い
  • for文はlistと同じように記述可能
import numpy as np

arr = np.array([4, 5, 6])

print(len(arr))  # 長さ（listと同じ）
print(arr.size)  # 普通はこちらを使う

# for文もlistと同じように使える
for v in arr:
    print(v)

# 差分（数値微分）
arr = np.array([0, 2, 5, 3, 1, 4]) # 時系列の信号を想定
diff = np.zeros(len(arr) - 1) # 結果格納用のndarray
for i in range(len(arr) - 1): 
    diff[i] = arr[i+1] - arr[i] # 微分
print(diff)  # [ 2.  3. -2. -2.  3.]

多次元Array

  • ndarrayは多次元（N-Dimensional）で力を発揮
    • 特に行列演算が強力
import numpy as np

# 2次元配列（行列）
A = np.array([[1, 2], [3, 4]])

B = np.array([[5, 0],
              [0, 5]])

# 要素へのアクセス
print(A[0, 1])  # 1行目2列目
print(A[1, :])  # 2行目全体
print(A[:, 0])  # 1列目全体

# 転置
print(A.T)
print(B.T)

その他演算等

  • 統計処理の組み込み関数も多数提供されている
import numpy as np

scores = np.array([80, 85, 90, 75, 88, 92, 82, 79, 87])

print(np.max(scores))    # 最大値
print(np.min(scores))    # 最小値
print(np.mean(scores))   # 平均値
print(np.median(scores)) # 中央値
print(np.var(scores))    # 分散
print(np.std(scores))    # 標準偏差

# 3人の生徒の英語・数学・国語の成績
# 多次元の処理はaxisを指定する
data = np.array([[80, 85, 90], [75, 88, 92], [82, 79, 87]])
print(np.mean(data, axis=0))  # 列ごと（各科目の平均）
print(np.mean(data, axis=1))  # 行ごと（各生徒の平均）

# 条件を満たすデータの抽出
print(scores >= 85)           # True/False
print(scores[scores >= 85])   # 85点以上のデータ
